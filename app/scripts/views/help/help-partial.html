Getting Started

MuseumStat is designed to be an intuitive yet powerful tool to provide important information on museums and communities. There are many ways to use MuseumStat to inform decision-making, understand community needs, and make a case for support. If you need any assistance using MuseumStat or would like to share feedback, please contact us at info@museumstat.org.

Search for Museums or Communities

Start by using the search bar to enter the name of a specific museum or enter an address, city, or zip code.

Analyze by Museum

Selecting a specific museum from the search bar will display a page for the museum selected and show its location on a map along with surrounding museums. A default radius of 1 mile is shown around the museum selected.

Below the map are important data on individuals and households in the area selected along with regional and statewide comparisons of museums. View data through the three tabs shown. Move your mouse over the charts to get more details of the data displayed.

Adjust the analysis area using the menu or by drawing a custom radius or shape. The data below will automatically update based on the area you selected.

Use the “Add Demographics” icon on the map to show important demographic measures on the map. Move your mouse over this layer to see the data on the demographic option selected.

Create a useful report of this map and data by selecting “Print This Report.” The report can be printed or saved as a PDF.

If you want to submit any updates to the information on a museum, select “Submit Museum Update” to contact us.

Analyze by Community

Selecting a specific address, city or zip code from the search bar will display the region selected and show its location on a map along with surrounding museums.

Below the map is a list of the surrounding museums that appear on the map.

Use the “View & Analyze” button next to a specific museum to get details of that museum along with important data on the surrounding community. See “Analyze by Museum” above for details.

From the list of museums, use the “Download Museum Info” link to download a file of all the museums listed along with relevant data on each museum.
