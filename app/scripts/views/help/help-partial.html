Getting Started

ImpactView Philadelphia uses an intuitive interface, combining maps, data visualization, and search functionality.

Search for Organizations

Explore the map by searching by organization, zip code, or address. If you enter a zip code or address the map will show that location with a list of organizations within that area. Select an organization by clicking on it in the map or from the list below for more details.

Analyzing Organizations

Explore the map by searching by organization, zip code, or address. If you enter a zip code or address the map will show that location with a list of organizations within that area. Select an organization by clicking on it in the map or from the list below for more details.

Once you have selected an organization, the map will show that organization on the map with a data visualization below comprising important measures of households and individuals using a default radius of 0.5 miles surrounding the organization.

Use the tool to change the radius for which the data is visualized.

Use the tool to filter by the type of nonprofit organization

Use the tool to view important demographics on the map.

Use the tool to maximize the view of the map for improved interaction.
